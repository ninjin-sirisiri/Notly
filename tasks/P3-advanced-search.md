# 高度な検索機能

## 概要
現在の基本的な検索機能を拡張し、より柔軟で強力な検索を実現します。

## 機能要件
1. **高度な検索オプション**
   - タイトルのみ/本文のみ/両方検索
   - 正規表現検索

2. **フィルタリング**
   - フォルダでフィルタ
   - タグでフィルタ
   - 日付範囲でフィルタ
   - お気に入りのみ表示

3. **検索結果の並び替え**
   - 関連性順/更新日順/作成日順/タイトル順

4. **検索ハイライト**
   - 検索結果のプレビューにキーワードをハイライト

## 実装手順
1. バックエンドの検索コマンド拡張（FTS5利用）
2. フィルタリング機能の実装
3. 検索結果UI改善
4. ハイライト機能追加
