# テンプレート機能の詳細化

## 概要
よく使うノート構造をテンプレートとして保存し、新規ノート作成時に利用できるようにします。

## 機能要件
1. **テンプレート作成**
   - 既存ノートからテンプレート作成
   - 手動でテンプレート作成

2. **テンプレート管理**
   - テンプレート一覧表示
   - 編集・削除

3. **テンプレート適用**
   - 新規ノート作成時にテンプレート選択
   - デフォルトテンプレート設定

## データベース設計
- `templates` テーブル（id, name, content, created_at, updated_at）

## 実装手順
1. データベーステーブル作成
2. バックエンドCRUD処理
3. テンプレート選択UI
4. 新規作成時の適用処理
