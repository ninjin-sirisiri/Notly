このディレクトリには、Notlyのノート・フォルダ機能に関する実装タスクがまとめられています。

## タスクファイル一覧

### 高優先度
- [P1-trash-feature.md](./P1-trash-feature.md) - ゴミ箱機能
- [P1-keyboard-shortcuts.md](./P1-keyboard-shortcuts.md) - キーボードショートカット

### 中優先度
- [P2-tag-system.md](./P2-tag-system.md) - タグ機能
- [P2-bookmark-feature.md](./P2-bookmark-feature.md) - お気に入り/ブックマーク機能

- [P3-folder-color.md](./P3-folder-color.md) - フォルダカラー設定

## 優先順位の基準

- **P1 (高優先度)**: ユーザビリティに直接影響し、実装によるメリットが大きい機能
- **P2 (中優先度)**: あると便利だが、なくても基本機能が使える機能
- **P3 (低優先度)**: 拡張機能やニッチなユースケース向けの機能

## タスクの進め方

1. 各タスクファイルを確認し、実装の詳細を確認する
2. 優先度に従って順次実装を進める
3. 実装完了後は、該当ファイルの「ステータス」を更新する
4. 必要に応じてタスクを分割または統合する
