# パフォーマンス最適化 (Performance Optimization)

## 概要
アプリケーションの動作速度とレスポンスを向上させ、ユーザー体験を改善する。

## 要件
- ノート数が増えた場合のリスト表示の最適化（仮想スクロールなど）
- エディタの入力遅延の最小化
- 起動時間の短縮
- メモリ使用量の削減

## タスク
- [ ] パフォーマンス計測・ボトルネックの特定
- [ ] リストコンポーネントの仮想化 (Virtualization)
- [ ] 不要な再レンダリングの抑制 (React.memo, useMemo, useCallback)
- [ ] データベースクエリの最適化
- [ ] バンドルサイズの削減
